# Security

repo-launch-skill is local-first and does not perform network writes. Please do not include private repository data, credentials, customer names, or unpublished launch plans in bug reports.

## Reporting

Open a GitHub security advisory or contact the maintainer privately for sensitive issues.

## Scope
- Secret leakage in generated output.
- Unsafe recommendations that imply external publication without approval.
- Fixture data that accidentally contains private material.
